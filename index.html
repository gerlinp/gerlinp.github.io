About Gerlin Pierre-Louis

Hi, I’m Gerlin — a software engineer who builds things for the web.

My path here wasn’t traditional. I started in IT, helping clients troubleshoot and navigate software — and somewhere along the way, helping people use tools turned into wanting to build better ones. I taught myself to code, made the switch, and haven’t looked back.

Today I work as a software engineer at ezCater, where I focus on frontend development with React and TypeScript. I’ve worked on design system migrations, large-scale codebase restructuring, and improving real user experiences — from checkout flows to SEO to code quality and observability.

I still care about what got me here: the idea that good technology should bridge gaps — between industries, between communities, and between what’s possible and what’s actually been built.

Every great developer you know got there by solving problems they were unqualified to solve until they actually did it.
